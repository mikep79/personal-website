Word of the day

String interpolation:

Process of evaluating a string literal containing placeholders (variables) replaced with values.

C# example

string breakfastFood = "waffles";

Console.WriteLine($"Pancakes are almost as tasty as {breakfastFood}.");

$ is used before a string, as a "heads up" to the compiler that the following string contains some logic! Our variable is then inserted using the { }. When I was learning JavaScript, we pretty much just concatenated our strings using , or +. The one instance where I used string interpolation I can think of is with AngularJS expressions in an html file:

<p>Here's a {{ vm.word }} example of string interpolation.</p>
