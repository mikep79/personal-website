To Do app

See the code on Github

Why I built it:

  • Helps you to keep track of things you need to get done!
  • I built this app using AngularJS and jQuery to have basic, full-stack, CRUD template to use for both frameworks.

What it does:

  • User creates tasks, and marks them as complete when finished.
  • Tasks can also be deleted.

This app was built with JavaScript, jQuery, AngularJS, PostgreSQL, Node.js, and Express.
