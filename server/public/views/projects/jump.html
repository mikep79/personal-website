JUMP - Join Up Meet People

Try out this app

See the code on Github

Why I built it:

  • JUMP encourages caregivers to share community resources or scheduled events sensitive to the needs of people with disabilities.

What it does:

  • Users create events tagged with accessibility and content needs, and share with other users.
  • Calendar of events can be filtered by content type or accessibility needs. For example, a user with hearing impairment may click on tags for deafness and movies to find events such as movie showings at theatres with closed captioning.
  • Users can register for an event and manage all their events.
  • Optimized for screen reader accessibility.

This app was built with the MEAN stack: Mongo Database, Express, Angular.js, Node.js.
