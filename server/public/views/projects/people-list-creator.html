People List Creator

See the code on Github

Why I built it:

  • My first useful jQuery app, this allows you to keep track of your acquaintances and their interesting traits!

What it does:

  • User adds a person's name and an interesting fact about them to a directory.
  • User can cycle through the people list with a carousel, and remove a person from the directory.

This app was built with JavaScript, jQuery, and Node.js.
