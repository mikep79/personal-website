Market Mania

See the code on Github

Why I built it:

  • Players can have fun managing money and posting high scores.

What it does:

  • Players buy and sell food items with $100 of starting money.
  • Prices change frequently. Buy low and sell high to gain the most amount of money within 2 minutes.
  • Players can register a username and password to post high scores. Game displays current high scorer.

This app was built with JavaScript, AngularJS, Node.js, Express, and MongoDB, and custom CSS styling.
