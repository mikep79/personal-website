Why I built it:

  • A fun way to share personal photos, this project was built to gain skill with AngularJS and deploying an app on Heroku.

What it does:

  • This app contains a gallery of photos that users can view, "like", or comment on.
  • Users can click on a photo to view a description, and view all comments.

This app was built with JavaScript, AngularJS, Node.js and custom CSS styling.
